# 开始

## 总览

Qutils 是公司内部孵化的 前端实用工具类库

你可以在 [什么是 Qutils](./why) 中了解有关该项目背后的基本原理的更多信息。

## 将 Qutils 安装到项目

```bash
# 使用 pnpm
pnpm add -D @qmai/qutils
# 使用 yarn
yarn add -D @qmai/qutils
# 使用 npm
npm install -D @qmai/qutils
```

:::tip 提示
部分功能需要配合第三方库使用。
:::
